# Aurum28 Core

Official Aurum28 (AUR28) blockchain core implementation.

Aurum28 is an independent SHA-256 Proof-of-Work blockchain designed as digital gold, with a fixed maximum supply of 28,000,000 coins and transparent, miner-secured infrastructure.

## Key Features

- Independent Layer-1 Blockchain
- SHA-256 Proof-of-Work Consensus
- Fixed Supply: 28,000,000 AUR28
- Open Mining (ASIC, GPU, CPU)
- Self-Custody Wallet Support
- Public Blockchain Explorer
- Transparent Monetary Policy
