# Security Policy (Aurum28)

## Reporting a Vulnerability
If you discover a security issue, please report it privately to:

Email: info@aurum28.org  
Subject: [SECURITY] Aurum28 Core Vulnerability Report

Please include:
- A clear description of the issue
- Steps to reproduce (if possible)
- Impact assessment
- Any suggested mitigation

## Responsible Disclosure
We follow responsible disclosure. Please do not publish details publicly until a fix is released.

## Supported Versions
This repository will publish tagged releases once the public source release is completed.
